# Local AI Projects — Ollama + Python + Node

This repository contains a complete, structured series of projects that teach you how to build powerful local AI applications using **Ollama**, **qwen3:14b**, **Python**, **Node.js**, and advanced agent tooling.

Each project builds on the previous one, gradually expanding your capabilities from simple API calls to a full AI IDE with agents, RAG, memory, and MCP.

---

## 📚 Project Index

### 01 — Ollama API  
Basic API usage with Python.

### 02 — Python Client  
Reusable Python chat client with config.

### 03 — Node Client  
Node.js chat client using axios.

### 04 — Terminal Chat  
Interactive terminal assistant with colors.

### 05 — Streaming  
Token-by-token streaming responses.

### 06 — Chat History  
Persistent conversation history stored in JSON.

### 07 — Memory  
Long-term memory system with JSON storage.

### 08 — RAG  
Document ingestion + vector DB + retrieval.

### 09 — File Reader  
AI that reads and explains local files.

### 10 — Git Tools  
AI assistant that interacts with Git.

### 11 — AI Agent  
Tool-based agent with prompts and actions.

### 12 — MCP  
Model Context Protocol server + client.

### 13 — AI IDE  
Full-stack AI coding environment (backend + frontend).
